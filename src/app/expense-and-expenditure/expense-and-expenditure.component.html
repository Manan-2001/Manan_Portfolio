Project Overview

Expense and Expenditure Application is a mobile-based financial management system designed to help users track daily expenses, manage spending habits, and monitor financial activities digitally. The system replaces manual expense tracking with a secure, structured, and automated solution.

Problem

  • Manual expense tracking
  • No expense categorization
  • No spending analytics
  • Poor financial planning

Solution

  • Digital expense tracking
  • Category-based expenses
  • Visual spending overview
  • Secure user accounts

Core Features

Secure Login
User Registration
Expense Tracking
Category Management
Add / Edit / Delete Expenses
Spending Overview
Firebase Authentication
Mobile Optimized UI

System Architecture

User → Android App (Kotlin) → Firebase Auth → Firebase Database → App Dashboard → User

My Role

I designed and developed the complete application including UI design, Kotlin-based development, Firebase authentication integration, database handling, Lottie animations, testing, and deployment as a full academic project.
