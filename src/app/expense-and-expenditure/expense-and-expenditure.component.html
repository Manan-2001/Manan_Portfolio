Expense and Expenditure Application


X





I developed an Expense and Expenditure Application as an academic project during my MCA program. The primary goal of this project was to create a user-friendly mobile application for tracking expenses, helping users manage their finances effectively.

The application was built using Android Studio and Kotlin, ensuring a modern and efficient development process. To enhance the user experience, I incorporated Lottie Animations, which provided smooth and visually appealing transitions within the app.

Key features of the Expense and Expenditure Application include functionalities for tracking daily expenses, categorizing expenditures, and providing an overview of spending patterns. Users can easily add, update, and delete expense entries, making it simple to manage and monitor their financial activities.

Additionally, the application includes secure login and registration activities. For user authentication, I integrated Firebase, which offers robust and reliable authentication services. This ensures that users' data is protected and only accessible to authorized individuals.

This project highlights my skills in mobile app development, particularly with Kotlin and Android Studio. It also demonstrates my ability to create intuitive and engaging user interfaces using Lottie Animations. Furthermore, the integration of Firebase for authentication underscores my competency in incorporating backend services to enhance application functionality and security.
