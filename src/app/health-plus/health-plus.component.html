Project Overview

Health Plus is a smart healthcare Android application designed to simplify medical services by allowing users to book doctor appointments, order lab tests, and manage health profiles through a secure and user-friendly mobile platform. The application transforms traditional healthcare processes into a digital, accessible experience.

Problem

  • Manual doctor appointment booking
  • No centralized health platform
  • Complex lab test scheduling
  • Poor user experience in health apps

Solution

  • Digital appointment booking
  • Centralized healthcare system
  • Integrated lab test ordering
  • Smooth mobile user experience

Core Features

Secure Login
Registration
Doctor Booking
Lab Test Orders
Appointment Scheduling

System Architecture

User → Android App → Firebase Auth → Firebase Database → Services → App UI → User
