During my internship at Ankhub Technologies Services, I had the opportunity to work on an innovative project aimed at developing a comprehensive rental services web application. This platform integrates four distinct rental services into a single seamless user experience: Accommodation Rent, Tiffin Rent, Vehicle Rent, and Electrical Appliances Rent.

Our team consisted of six members, and we utilized a variety of technologies to bring this project to life. On the frontend, we employed Angular for building dynamic and responsive user interfaces, along with HTML, CSS, and Bootstrap for designing a user-friendly and visually appealing layout. JavaScript and TypeScript were used to enhance the functionality and interactivity of the application. For version control and collaborative development, we relied on Bitbucket and GitHub.

My primary responsibilities included frontend development, where I played a key role in creating and optimizing the user interface. I worked on various features across the four rental categories. For Accommodation Rent, I developed features allowing users to search, filter, and book rental properties based on their preferences and location. In the Tiffin Rent module, I enabled users to subscribe to daily meal services with customizable meal plans. For Vehicle Rent, I implemented a system for users to book and schedule vehicle rentals, complete with payment functionalities. In the Electrical Appliances Rent section, I developed a user-friendly interface for renting appliances, including detailed descriptions and rental terms.

Ensuring a seamless user experience was a top priority. I focused on making the web application fully responsive, providing a consistent experience across different devices and screen sizes. My work involved optimizing the user interface for ease of navigation and improved user satisfaction.

Additionally, I was actively involved in testing and debugging the application. I conducted thorough testing to ensure reliability and performance, working closely with my team to resolve issues and implement improvements based on feedback.

The successful completion of this project resulted in a robust and user-friendly web application that streamlined multiple rental services into a single platform. This integration provided users with a convenient and efficient solution for their rental needs.

Through this project, I gained valuable hands-on experience in Angular and modern web development practices. I also improved my skills in teamwork, collaboration, and version control, enhancing my problem-solving abilities through debugging and optimizing web applications. This project not only honed my technical skills but also provided valuable insights into developing comprehensive web solutions in a collaborative environment.
