Sports Activities Recognition System


X





I developed a Sports Activity Recognition System as an academic project during my MCA program. The objective of this project was to train a model capable of predicting sports activities from input sports videos and images. This project employed advanced machine learning and deep learning techniques to achieve high accuracy in sports recognition.

To build the model, I utilized the ResNet-50 deep learning algorithm, a powerful convolutional neural network known for its efficiency in image and video classification tasks. The model was trained using a comprehensive Sport Image Dataset, which provided a wide range of sports activities for accurate classification. Additionally, I used TensorFlow as the primary framework for implementing and training the model.

The system incorporates Label Binarizer for encoding the sports labels, enabling the model to differentiate between various sports activities effectively. The model can predict the type of sport from both images and videos, making it versatile for different input types. Throughout the training process, extensive image and video classification techniques were applied to enhance the model's performance.

The Sports Activity Recognition System achieved a remarkable training accuracy of 96.29%, indicating its proficiency in accurately identifying sports activities from input data. This high level of accuracy underscores the effectiveness of the ResNet-50 algorithm and the robustness of the dataset used.

This project provided valuable hands-on experience in applying deep learning algorithms to real-world problems, showcasing my ability to leverage advanced technologies for developing intelligent systems. It also demonstrated my skills in data preprocessing, model training, and performance evaluation, essential for any machine learning and deep learning project.
